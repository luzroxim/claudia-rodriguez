# Exploración de materiales

Este tutorial comparte mi trabajo con fibras, cartones y materiales locales para encontrar nuevas texturas.

## Materiales usados

- papel y cartón reciclado
- MDF y placas ligeras
- telas con distintas densidades

## Proceso

Documenté cada prueba para entender cómo reacciona cada material al corte, al plegado y al pegado.

![Materiales](../images/proyecto/8.jpg)

## Observaciones

El desafío principal fue equilibrar la flexibilidad con la rigidez, y construir fragmentos que funcionen como superficie y como soporte.
